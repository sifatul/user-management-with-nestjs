## Description

Online step by step learning of [Nest](https://github.com/nestjs/nest) framework with TypeScript.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
